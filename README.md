## ğŸš€ Getting Started

### Prerequisites

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd machine_coding
   ```

2. **Install dependencies**
   ```bash

   ```

3. **Run the app**
   ```bash
   # For iOS
   npx react-native run-ios

   npx react-native run-android
   ```
